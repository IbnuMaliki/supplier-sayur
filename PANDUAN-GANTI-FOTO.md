## 🚀 Cara Ganti Foto (4 Langkah)

### Langkah 1 — Login Admin
```
Buka : http://localhost/web_programing%201/supplier-sayur/login.php
Email: admin@suppliersayur.com
Pass : admin123
```

### Langkah 2 — Buka Menu Dokumentasi
```
Klik nama Anda (pojok kanan atas)
→ Admin Panel
→ Klik menu 🖼️ Dokumentasi (di sidebar kiri)
```

### Langkah 3 — Pilih Foto yang Ingin Diganti
```
Lihat tabel daftar foto
→ Klik tombol ✏️ Ganti Foto
→ Form edit muncul di bawah halaman
```

### Langkah 4 — Upload Foto Baru
```
Klik kotak upload besar
→ Pilih foto dari komputer/HP Anda
→ Lihat preview foto yang dipilih
→ Isi Judul (wajib) & Keterangan (opsional)
→ Klik 💾 Simpan Perubahan
→ Buka homepage → foto sudah terganti! ✅
```

## ➕ Cara Tambah Foto Baru

```
Admin Panel → 🖼️ Dokumentasi
→ Scroll ke bagian bawah (form "➕ Tambah Foto Baru")
→ Upload foto → isi Judul → isi Urutan → Centang "Tampilkan di Homepage"
→ Klik 📤 Upload & Tambahkan
```

## 📋 Aturan Foto

| Hal          | Keterangan                                      |
|--------------|-------------------------------------------------|
| Format       | JPG, PNG, WEBP (bukan GIF atau PDF)            |
| Ukuran file  | Maksimal **5 MB** per foto                      |
| Jumlah tampil| Maksimal **5 foto aktif** di homepage           |
| Urutan 1     | Posisi **kiri besar** (foto paling menonjol)   |
| Urutan 2–5   | Posisi **kanan kecil**                          |

---

## 🔧 Fitur Tambahan

| Tombol/Fitur        | Fungsi                                        |
|---------------------|-----------------------------------------------|
| ✅ Tampil / ⭕ Disembunyikan | Sembunyikan foto sementara tanpa hapus |
| 🗑️ Hapus           | Hapus foto permanen dari server               |
| Drag & Drop         | Seret foto langsung ke kotak upload           |

---

## ❓ Troubleshooting

**Upload gagal / foto tidak berubah?**
- Pastikan ukuran file ≤ 5 MB
- Gunakan format JPG, PNG, atau WEBP
- Pastikan XAMPP Apache & MySQL sudah Start

**Foto muncul di tabel tapi tidak di homepage?**
- Cek status: harus **✅ Tampil** (bukan ⭕ Disembunyikan)
- Cek urutan: hanya 5 foto urutan terkecil yang tampil

**Foto lama masih muncul setelah ganti?**
- Tekan **Ctrl + F5** di browser untuk refresh paksa (clear cache)
